# Sales Performance Dashboard (Power BI)

## Project Overview

This project presents a Sales Performance Dashboard built using Power BI and the Superstore dataset. The dashboard provides insights into sales, profit, quantity sold, regional performance, product performance, and category-wise analysis. It helps businesses monitor key performance indicators (KPIs) and make data-driven decisions.

## Features

* Total Sales Analysis
* Total Profit Analysis
* Quantity Sold Tracking
* Sales Trend Analysis (Year-wise)
* Sales by Category
* Profit by Category
* Regional Sales Distribution
* Top-Selling Products Analysis
* Interactive Filters (Region, Category, Ship Mode, Order Date)

## Tools & Technologies

* Power BI Desktop
* Microsoft Excel
* Data Visualization
* Business Intelligence (BI)

## Dashboard Insights

* Technology category generates the highest profit.
* Sales show a steady growth trend over the years.
* Regional analysis highlights the strongest performing markets.
* Product-level analysis identifies top revenue-generating products.
